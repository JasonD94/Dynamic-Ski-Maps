Dynamic Trail Maps

This site will host links to Dynamic Ski Maps, which are maps that will display open / closed trail & ski lifts for the given Ski Resort.

Currently a "working" demo for Waterville Valley is available, with plans for Bretton Woods and other resorts to be added later.
